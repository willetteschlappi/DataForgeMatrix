# DataForgeMatrix - Structured Dataset Generator

DataForgeMatrix creates synthetic datasets for testing APIs, UIs, machine learning, and backend systems.

## Features
- Output formats: CSV / JSON / SQL  
- Supports nested fields  
- Fast and scriptable
